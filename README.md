# Project Name: Nature Explorer India
## Description: A simple 4-page website showcasing top nature destinations in India.

**How to Run:**
1. Save each block of code into its respective filename (e.g., index.html).
2. Ensure all files are in the same folder.
3. Open 'index.html' in any web browser.

Pages:
- index.html: Introduction and main navigation.
- menu.html: A list of top sites presented in a table format.
- about.html: Background information on the project mission.
- contact.html: A functional-looking form for user inquiries.
